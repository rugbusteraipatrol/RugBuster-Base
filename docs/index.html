// CIA Engine

Five Modules, One Readout

The Base collector keeps the same CIA Engine structure as the existing RugBuster EVM workers. It does not guess safety from hype; it combines chain facts, liquidity context, and behavioral signals.

Funding Origin

Tracks deployer funding hops and fresh-wallet patterns around the token launch.

Holder Cluster

Checks concentration risk and early distribution shape when holder data is available.

Backdoor Check

Inspects bytecode for owner, mint, pause, blacklist, proxy, and drain-style function selectors.

Liquidity Status

Reads Base pool evidence from supported DEX sources and market-data feeds.

Final Verdict

Combines module scores into GOOD, WARN, or DANGER, with reasons stored in Postgres.
